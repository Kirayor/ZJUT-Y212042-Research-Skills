# OpenClaw Research Skills

This repository collects research-oriented OpenClaw skills developed by graduate students of Class Y212042, 2026 Summner, ZJUT.

## Goal

The goal of this project is to build reusable agent skills for real research workflows, including experiment diagnosis, dataset auditing, reviewer response preparation, reproducibility analysis, claim-evidence checking, and research project management.
