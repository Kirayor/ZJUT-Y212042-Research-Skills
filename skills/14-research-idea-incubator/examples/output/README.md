# Example Output: Diffusion-Edited VLM Robustness Idea

This example records one completed run of the Research Idea Incubator workflow.

Input idea:

> Use diffusion models for adversarial example generation in visual-language model security.

Generated artifacts:

- `idea_card.md`: clarified scope and assumptions.
- `keywords.md`: bilingual keyword groups and search queries.
- `paper_candidates.md`: OpenAlex-based candidate paper list with PDF links where available.
- `paper_summaries.md`: first-pass paper summaries, marked for manual verification.
- `literature_matrix.md`: comparison matrix across task, method, contribution, limitation, and relation to the idea.
- `research_question.md`: final recommended research question and minimal experiment plan.
- `process_data.json`: dashboard state for visualizing the workflow.

The paper summaries in this example should be treated as workflow evidence, not final scholarly claims. They intentionally keep manual-verification warnings where full-text reading is still needed.
